Jugando con git

Que es git

Git es un software de control de versiones open source disenado para manejar desde pequenos hasta enormes proyectos con velocidad y eficiencia.

Por que utilizarlo

Nos ayudara a llevar un historial completo de las modificaciones de nuestro proyecto.

Podriamos pensar como si fuera una maquina del tiempo y poder viajar en el tiempo dentro de nuestro proyecto.

Podremos hacer bifurcaciones para no compormeter la rama principal de esta forma lanzar versiones simultaneamente.

Trabajar en equipo siempre es vital y git permite hacerlo.

Todos los colaboradores tienen una copia local del proyecto y tambien el repositorio remoto contiene todo.(distribuido)
